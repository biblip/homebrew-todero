# Homebrew Tap for Todero

This repository is the official Homebrew tap for **Todero**.

It exists solely to make Todero easy to install on macOS using Homebrew.  
The formulae in this repository are automatically updated when new versions
of Todero are released from the main project repository.

## Install

Add the tap:

    brew tap biblip/homebrew-todero

Install one of the available packages:

    brew install todero
    brew install todero-jvm-engine
    brew install todero-jvm-runner

## Notes

- This repository contains Homebrew formulae only.
- Source code, issues, feature requests, and development discussions belong in
  the main Todero repository.
