## Contribution Expectations

While the Apache License permits unrestricted forking,
Todero strongly encourages improvements to be proposed
upstream via pull requests.

Long-lived forks are discouraged unless they serve a clear
compatibility, experimentation, or research purpose.

This repository represents the canonical implementation
of the Todero platform.
